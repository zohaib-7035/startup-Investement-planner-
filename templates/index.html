VC Brain — Agentic Founder Intelligence · Local & Offline-Safe

Agentic Founder Intelligence

VC Brain is a fully local, offline-safe multi-agent pipeline designed to discover, screen, and verify investment claims for startup founders using dev signals.

Project Architecture & Workflow
1️⃣
Thesis Config

Define target sectors, funding stages, geographic filters, risk tolerances, and check size parameters.

2️⃣
Founder Sourcing

Consolidate inbound pitch submissions and outbound developer profiles crawled automatically from GitHub repos.

3️⃣
Agentic Screening

Evaluate profiles on three independent axes (Founder, Market, and Fit) using LLM reasoning rules.

4️⃣
Fact Verification

Extract core founder claims and verify them rules-based against developer signals to detect contradictions.

5️⃣
Investment Memo

Synthesize findings into an evidence-backed report with a calculated Overall Trust Score and print support.

Agent Architecture

VC Brain operates on a multi-agent orchestration architecture that divides intelligence concerns:

  • Sourcing Agent: Scans GitHub repositories evaluating commit histories, stack depth, and team contributors.
  • Screening Agent: Screens candidates independently against custom thesis parameters.
  • Trust Verification Agent: Uses semantic extraction to verify claims against objective developer logs.
  • Report Generation Agent: Structures summaries, marking gaps in knowledge as "not disclosed".
Traceability & Safety

Engineered from the ground up for critical investment environments where black-box decision models fail:

  • Zero Hallucinations: The pipeline explicitly states details as "not disclosed" if information is absent, preventing data fab.
  • Agentic Auditing: The multi-step reasoning log exposes the exact step and confidence value behind each conclusion.
  • Independent Axes: The three analysis scores are never compounded into a single metric, keeping indicators clean.
  • Local and Secure: Fully offline-capable design running local models via Ollama to keep IP safe.
